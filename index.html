Cybersecurity Labs

This lab demonstrates how to configure WampServer, identify SQL injection vulnerabilities, and exploit PowerShell for fileless malware attacks.

Lab 2: Social Engineering Tools and Phishing Techniques.

This lab explores tools like SET, BeEF, and phishing techniques to simulate social engineering attacks and evaluate browser vulnerabilities.

Lab 3: Exploiting Application Vulnerabilities using ZAP, XSS, and URL Manipulation.

Demonstrates the exploitation of application vulnerabilities such as XSS, insecure direct object references (IDOR), and HTTP request manipulation using OWASP ZAP.

Lab 4: Capturing Hashes, Brute Forcing, and Wireless Testing.

This lab demonstrates techniques for capturing NTLM hashes, brute-forcing credentials, and analyzing wireless networks using tools like Responder, Hydra, NetSpot, and Advanced IP Scanner.

Lab 5: Exploiting Host Vulnerabilities using SAM, Hashcat, and Shell.
